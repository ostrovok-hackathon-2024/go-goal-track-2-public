# Tagger python

## Installation

```sh
poetry install
```

## CLI

```sh
poetry run cli --help
```

## API

```sh
poetry run server
```

## Configuration

```yaml
MODELS_DIR: ./models
CATEGORIES:
  - category1
  - category2
```

## Benchmark

```sh
./bench/benchmark.sh
```

## Tests

```sh
pytest
```

## Linter

```sh
pylint .
```
